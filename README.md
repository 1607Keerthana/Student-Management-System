# Student Management System

## Objective

A console-based Java application to manage student records using Object-Oriented Programming concepts.

## Features

* Add Student
* View All Students
* Search Student by ID
* Search Student by Name
* Update Student Grade
* Delete Student
* Calculate Average Grade
* Find Top Performer
* Save Data to File

## Technologies Used

* Java
* IntelliJ IDEA
* ArrayList
* File Handling
* OOP Concepts

## How to Run

1. Open project in IntelliJ IDEA
2. Run Main.java
3. Use menu options to manage students
